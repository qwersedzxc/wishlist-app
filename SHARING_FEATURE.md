# Функционал "Поделиться вишлистом"

## Описание

Добавлена возможность делиться ссылками на вишлисты с друзьями через систему уведомлений.

## Что реализовано

### 1. Кнопки "Поделиться" на странице вишлиста

На странице просмотра вишлиста (`view_wishlist.php`) добавлены кнопки:

- **"Скопировать ссылку"** - копирует прямую ссылку на вишлист в буфер обмена
- **"Отправить друзьям"** - открывает отдельное окно браузера для выбора друзей

### 2. Окно выбора друзей

При нажатии на "Отправить друзьям" открывается новое окно браузера (popup) с:
- Названием вишлиста
- Списком всех друзей (с чекбоксами для выбора)
- Полем для добавления сообщения
- Кнопками "Отправить" и "Отмена"

Размеры окна: 600x700px, по центру экрана

### 3. Система уведомлений

Создана таблица `notifications` для хранения уведомлений:

```sql
CREATE TABLE notifications (
    id SERIAL PRIMARY KEY,
    user_id INTEGER NOT NULL,
    type VARCHAR(50) NOT NULL,
    title VARCHAR(200) NOT NULL,
    message TEXT,
    link VARCHAR(500),
    from_user_id INTEGER,
    is_read BOOLEAN DEFAULT FALSE,
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);
```

### 4. API Endpoints

Добавлены новые endpoints в `api.php`:

- `GET /api.php?action=get_friends` - получить список друзей
- `POST /api.php?action=share_wishlist` - отправить вишлист друзьям
- `GET /api.php?action=get_notifications` - получить уведомления
- `POST /api.php?action=mark_notification_read` - отметить уведомление как прочитанное
- `POST /api.php?action=mark_all_notifications_read` - отметить все как прочитанные

### 5. Модель Notification

Создан класс `Notification` (`src/models/Notification.php`) с методами:
- `create()` - создание уведомления
- `getByUserId()` - получение уведомлений пользователя
- `getUnreadCount()` - количество непрочитанных
- `markAsRead()` - отметить как прочитанное
- `markAllAsRead()` - отметить все как прочитанные
- `delete()` - удалить уведомление

## Как использовать

### Для владельца вишлиста:

1. Откройте свой вишлист
2. Найдите секцию "Поделиться вишлистом"
3. Выберите один из вариантов:
   - **Скопировать ссылку** - для отправки вручную
   - **Отправить друзьям** - переход на страницу отправки через систему уведомлений

### Отправка друзьям:

1. Нажмите "Отправить друзьям"
2. Откроется страница выбора друзей
3. В форме выберите друзей (можно несколько)
4. Добавьте сообщение (необязательно)
5. Нажмите "Отправить"
6. Страница покажет результат отправки

### Для получателя:

1. Получите уведомление о том, что друг поделился вишлистом
2. Перейдите во вкладку "Друзья" - там будет секция "Полученные вишлисты"
3. В навигации рядом с "Друзья" появится красный индикатор с количеством непрочитанных уведомлений
4. Нажмите "Посмотреть" чтобы перейти к вишлисту
5. Отметьте уведомление как прочитанное или отметьте все сразу
